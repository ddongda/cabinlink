---
description: 新建一份架构决策记录（ADR）
argument-hint: <决策主题>
---

为「$ARGUMENTS」新建一份 ADR：

1. 读 docs/20-design/adr/ADR-0000-模板.md 和已有 ADR 确定下一个编号；
2. 按模板写入 docs/20-design/adr/ADR-<编号>-<主题>.md，
   必须包含：背景 / 至少两个被否选项及否决理由 / 决策 / 代价 / 推翻条件；
3. 「代价」段不允许为空——没有代价的决策记录无效；
4. 更新 docs/90-knowledge/INDEX.md（如该决策影响工程当前位置描述）。
